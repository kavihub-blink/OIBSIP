# OIBSIP
# Email Spam Detection

## Description
This project detects spam emails using natural language processing and machine learning. It helps users filter unwanted emails efficiently and understand patterns in spam content.

## Features
- Text preprocessing (tokenization, stopword removal)
- Feature extraction using TF-IDF
- Machine learning models (Naive Bayes, SVM)
- Accuracy evaluation and prediction on new emails

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- NLTK, Matplotlib
